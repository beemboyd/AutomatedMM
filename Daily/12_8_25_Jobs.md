### Job Status Codes

| Status | Meaning |
|--------|---------|
| `0` | Last run succeeded |
| `-` | Not run yet / Stopped |
| `1`, `2` | Exit with error code |
| `PID` | Currently running (shows process ID) |

## Job Management Commands

### View All Loaded Jobs
```bash
launchctl list | grep india-ts
```

### Check Specific Job Status
```bash
launchctl list | grep "job-name"
```

### Start a Job Manually
```bash
launchctl start com.india-ts.job-name
```

### Stop a Job
```bash
launchctl stop com.india-ts.job-name
```

### Load a Job (Register)
```bash
launchctl load ~/Library/LaunchAgents/com.india-ts.job-name.plist
```

### Unload a Job (Deregister)
```bash
launchctl unload ~/Library/LaunchAgents/com.india-ts.job-name.plist
```

### Reload a Job (After Editing)
```bash
launchctl unload ~/Library/LaunchAgents/com.india-ts.job-name.plist
launchctl load ~/Library/LaunchAgents/com.india-ts.job-name.plist
```

### Install All Jobs from Backup
```bash
python /Users/maverick/PycharmProjects/India-TS/Daily/scheduler/install_plists.py
```

### Validate All Jobs
```bash
python /Users/maverick/PycharmProjects/India-TS/Daily/scheduler/validate_plists.py
```

### View Job Logs
```bash
# Most job logs are in Daily/logs/
tail -f /Users/maverick/PycharmProjects/India-TS/Daily/logs/job_name.log
```

## Troubleshooting Jobs

### Job Not Running
```bash
# Check if loaded
launchctl list | grep job-name

# Check plist syntax
plutil -lint ~/Library/LaunchAgents/com.india-ts.job-name.plist

# Check logs
tail -100 ~/Library/Logs/com.india-ts.job-name/stderr.log
```

### Job Keeps Crashing
```bash
# Check exit code
launchctl list | grep job-name
# Non-zero exit code indicates error

# Check error log
cat /Users/maverick/PycharmProjects/India-TS/Daily/logs/job-name_error.log
```

### Job Not Starting at Scheduled Time
```bash
# Verify schedule in plist
plutil -p ~/Library/LaunchAgents/com.india-ts.job-name.plist | grep -A10 StartCalendarInterval

# Check system logs
log show --predicate 'subsystem == "com.apple.xpc.launchd"' --last 1h | grep job-name
```
